# Epickie Wakacje — Włochy 2026

Interaktywny planer rodzinnego wyjazdu do Toskanii (23.07–05.08.2026).

Aplikacja to jeden samodzielny plik **`index.html`** — działa w każdej przeglądarce, bez logowania i bez internetu. Dane (plan dnia, wydatki) zapisują się lokalnie w przeglądarce.

## Zakładki
- **Plan** — planer dnia z mapą trasy. Dni przejazdowe **23.07 (postój w Brnie)** i **24.07 (Wenecja)** mają rozpisany plan z godzinami, parkingiem i restauracjami. Minione dni się blokują, a wybrane wycieczki „zużywają się".
- **Wycieczki** — gotowe scenariusze do wstawienia w plan
- **Propozycje** — dodatkowe kierunki (Siena, Cinque Terre, Pienza)
- **Bilety** — linki do oficjalnych kas
- **Na lody & dzieci** — lżejsze popołudnia
- **Koszty** — kalkulator budżetu + wydatki na bieżąco per rodzina

## Publikacja przez GitHub Pages
1. Wgraj `index.html` (i ten `README.md`) do repozytorium.
2. **Settings → Pages → Deploy from a branch → main / (root) → Save**.
3. Strona pojawi się pod `https://<nazwa-uzytkownika>.github.io/<repo>/`.

## Aktualizacja
Po każdej zmianie podmieniasz na GitHubie plik `index.html` (ta sama nazwa nadpisuje stary). Strona przebuduje się sama w 1–2 min, a link pozostaje ten sam.
